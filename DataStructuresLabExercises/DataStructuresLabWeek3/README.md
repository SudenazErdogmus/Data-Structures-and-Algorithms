VERİ YAPILARI LABORATUVARI HAFTA-3

1. CircularQueue ve GenericStack sınıflarını projeye dahil ediniz. 
2. CirculerQueue sınıfı içinde count methodu oluşturun. Bu method Queue’daki eleman 
sayısını geri döndürmelidir. 
3. Test class’ı içinde bir search methodu oluşturun. Parametre olarak CircularQueue ve 
target değeri almalıdır. target değerinin Queue içinde olup olmadığını bulmalıdır. Var 
ise true değilse false dönmelidir. (CompareTo methodu ile gerçekleştiriniz.) 
4. Test class’ı içinde bir isPalindrome methodu oluşturun. Parametre olarak bir string 
almalıdır. String palindrom ise true değilse false dönmelidir. GenericStack ve 
CircularQueue kullanarak gerçekleştiriniz.
